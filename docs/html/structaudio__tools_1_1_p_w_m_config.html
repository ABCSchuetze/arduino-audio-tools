int audio_tools::PWMConfig::resolution = 8

Configuration for PWM output.

RES BITS MAX_FREQ (kHz)
8 256 312.5
9 512 156.25
10 1024 78.125
11 2048 39.0625

The default resolution is 8. The value must be between 8 and 11 and also drives the PWM frequency.
